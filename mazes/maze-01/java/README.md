Running the Program
------------------- 

Fist compile the Driver.java file using "javac". 

To run the maze with default settings: 
<pre>
> java Driver
</pre>

To run the maze with custom width and height: 
<pre>
> java Driver -w20 -h25
</pre>

To run the maze with a preset seed, to model deterministic behavior: 
<pre>
> java Driver -s100
</pre>
